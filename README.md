# FWI ML Flask App

This is a Machine Learning web application built using Flask.

## Features
- Predicts Fire Weather Index (FWI)
- Uses Ridge Regression model
- Web interface for user input

## Tech Stack
- Python
- Flask
- Scikit-learn

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python application.py

3. Open browser:
   http://127.0.0.1:5000
